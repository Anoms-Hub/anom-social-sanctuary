# Anom Artsy - The Official AO Sanctuary

Welcome to the official code repository for the AO Sanctuary. This project serves as the digital home base for cyberpunk digital art, AI-driven storytelling, and family-friendly animation series.

## 🌟 About the Project

The AO Sanctuary is designed to be a safe, vibrant community hub. It combines neon-dark aesthetics (deep blacks, hot magenta, and neon cyan) with modern web technologies to deliver a seamless experience for showcasing digital assets and animated adventures.

### Core Features
* **Digital Art Gallery:** A showcase of custom AI-prompted artwork and profile skins.
* **Family-Friendly Content:** Home to the Clifford and Tater snack adventures and Anom's Corner.
* **Kid-Safe Environment:** Designed with community safety and positive social impact in mind.
* **Modern Web Architecture:** Built for speed and smooth performance.

## 🛠️ Tech Stack

This project is built using:
* **Frontend Framework:** React + TypeScript
* **Build Tool:** Vite
* **Styling:** Custom CSS (Neon Cyberpunk Aesthetic)
* **Deployment & AI:** Manus Space Integration / GitHub
